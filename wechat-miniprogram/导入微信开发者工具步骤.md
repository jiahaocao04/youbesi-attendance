# 导入微信开发者工具步骤

当前电脑上我没有检测到“微信开发者工具”，只检测到普通微信。普通微信不能导入小程序工程，必须安装微信开发者工具。

## 第一步：安装微信开发者工具

官方下载页：

```text
https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
```

下载安装 Windows 稳定版即可。

## 第二步：导入项目

打开微信开发者工具后：

1. 点击“导入项目”。
2. 项目目录选择：

```text
C:\Users\jhjhjh\OneDrive\文档\净雅中心广告\wechat-miniprogram
```

3. AppID：
   - 免费内测/看界面：可选测试号/游客模式。
   - 正式发布：以后再换成正式小程序 AppID。

4. 项目名称可以填：

```text
优贝思点名
```

## 第三步：登录小程序

开发者工具里先用：

```text
服务器：http://127.0.0.1:8766
用户名：admin
PIN：123456
```

如果用真机预览，服务器不能填 `127.0.0.1`，要填电脑局域网地址，例如：

```text
http://192.168.1.23:8766
```

电脑和手机必须在同一个 Wi-Fi 下。

## 目前已经做好的小程序功能

- 登录
- 统一点名册
- A/P 点名
- 事件提醒
- 学生购物：物品 + 金额
- 未到提醒
- 保存到现有系统数据库

## 正式上线前还需要

- 正式微信小程序 AppID
- HTTPS 域名
- 云服务器
- 数据库备份
- 微信后台配置合法域名
