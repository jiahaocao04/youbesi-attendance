# 优贝思点名微信小程序（内测免费版）

这是老师端小程序骨架，先用于免费内测和界面确认。

## 怎么打开

1. 安装并打开“微信开发者工具”。
2. 选择“导入项目”。
3. 项目目录选择：

```text
C:\Users\jhjhjh\OneDrive\文档\净雅中心广告\wechat-miniprogram
```

4. AppID 暂时使用测试/游客模式即可。

## 当前功能

- 登录
- 自动载入统一点名册
- A/P 点名
- 事件提醒
- 学生购物：物品 + 金额
- 保存到现有后端
- 未到提醒子页面

## 免费内测注意

- 开发者工具里可以用 `http://127.0.0.1:8766`。
- 真机预览时，手机不能访问电脑自己的 `127.0.0.1`，需要填电脑的局域网地址，例如：

```text
http://192.168.1.23:8766
```

- 正式发布微信小程序时，需要换正式 AppID、HTTPS 域名和云端服务器。
